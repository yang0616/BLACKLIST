# Blacklist
  此项目先利用MVC模型将系统功能鲜明分层后，配合类的单例模式先项目的框架构造出来，针对在公共或游玩娱乐等场所，有不雅或违背规章制度的人名单限制。
### 功能 ###
 将用户信息以及被限制人信息添加到各自对应的链表中进行添加、删除、修改、查找操作。链表节点中存储限制时长、行为等有关内容的记录。
